# Anki template: FB

A plain and simple template, in imitation of Facebook's FB5 design.

![Template Snapshot](https://i.imgur.com/ShC0fD5.png)

<br>

## Features

- No third-party dependencies. No font or images required.
- Animation: The answer fades in when revealed.

![GIF](https://i.imgur.com/QzTCLZE.gif)

- Night mode/Dark mode is supported.

![Imgur](https://i.imgur.com/O3kc3I5.png)
